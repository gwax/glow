class GlowLabeledPopupMenuWidget

General information

type: class
inherits: GlowPopupMenuWidget, GlowWidgetLabelHelper
module: glowMenuButtonWidget

GlowLabeledPopupMenuWidget is a convenience class of popup menu widgets that includes a label. It also packs the label appropriately.

GLOW programs that require labeled popup menus may use this class.
