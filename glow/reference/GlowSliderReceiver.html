typedef GlowSliderReceiver

General information

type: abstract
synonym: TReceiver<const GlowSliderMessage&>
module: glowSliderWidget

GlowSliderReceiver is the base class of receivers of indicator dragged and released events. Subclasses should override the OnMessage() method to handle such an event.

Most GLOW programs that use sliders will use GlowSliderReceiver to handle slider events.

Types

Types

typedef const GlowSliderMessage& MessageType

Methods

Destructor

~GlowSliderReceiver(void)

Automatically informs any attached GlowSliderWidgets that this receiver is going away.

Information on senders
A receiver can provide rudimentary information on which senders are bound to it.

unsigned int NumSenders(void)

Returns the number of scroll bars bound to this receiver.

unsigned int NumTrackers(void)

Returns the number of trackers bound to this receiver.

Overrideable methods

Receiving messages

virtual void OnMessage(const GlowSliderMessage& message) = 0

This is a pure virtual method that must be overridden by a subclass. It is called whenever the receiver receives an indicator dragged event.
