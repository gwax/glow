General information

type: struct
inherits: (none)
module: glow

GlowIdleMessage encloses data sent to a GlowIdleReceiver to notify it of an idle event.

GLOW programs that use idle events will need to use GlowIdleMessages.

Data

Event data

No data is sent with an idle event.
