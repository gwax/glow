module glowPushButtonWidget

General information

The glowPushButtonWidget module contains classes that implement pushbutton widgets.

Classes

class GlowPushButtonWidget

GLOW pushbutton widget.

class GlowDismissPushButtonWidget

GLOW pushbutton widget that dismisses its parent toplevel window.

class GlowWidgetMapToPushButtonFilter

Event filter that maps keypresses to hit a specified pushbutton.

struct GlowPushButtonParams

Parameter block for creating a GLOW pushbutton.

struct GlowPushButtonMessage

Message sent by a GLOW pushbutton.

abstract GlowPushButtonReceiver

Base class for receivers of pushbutton toggle events.
