class GlowLabeledSliderWidget

General information

type: class
inherits: GlowSliderWidget
module: glowSliderWidget

GlowLabeledSliderWidget is a numeric slider widget that includes a label. It supports both vertical and horizontal sliders, either ascending or descending, with either linear or logarithmic scale. It also packs the label appropriately.

GLOW programs that make use of labeled slider widgets should use this class.
