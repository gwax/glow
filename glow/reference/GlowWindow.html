class GlowWindow

General information

type: class
inherits: GlowSubwindow
module: glow

GlowWindow is the base class of toplevel windows. It extends GlowSubwindow by adding the concepts of iconification and title. A GlowWindow is at the top of every component hierarchy.

All GLOW programs must make use of toplevel windows.

Constants

Special window positioning constant

enum { autoPosition }

Allow the window manager to position the window as it chooses.

Methods

Constructors and destructor

GlowWindow(const GlowWindowParams& params)

Creates a new GlowWindow with options specified by the given params. Also creates the corresponding GLUT window.

GlowWindow(const char* title, int x, int y, int width, int height, Glow::BufferType mode, Glow::EventMask eventMask)

Creates a new GlowWindow with options specified by the given params. Also creates the corresponding GLUT window. title denotes the title of the window. x and y denote the pixel location of the top left corner of the window, in screen pixel coordinates. The constant GlowWindow::autoPosition may be used instead. width and height specify the width and height in pixels. mode describes the type of frame buffer to use. eventMask specifies which events the window wishes to receive when active. Use the event constants described in Glow for the event mask.

GlowWindow(void)

Creates a new GlowWindow but does not initialize it. A window created in this way may not be used until its Init() method is called.

void Init(const GlowWindowParams& params)

Initializes the GlowWindow with options specified by the given params. Also creates the corresponding GLUT window.

void Init(const char* title, int x, int y, int width, int height, Glow::BufferType mode, Glow::EventMask eventMask)

Initializes the GlowWindow with options specified by the given params. Also creates the corresponding GLUT window. title denotes the title of the window. x and y denote the pixel location of the top left corner of the subwindow, in screen pixel coordinates. width and height specify the width and height in pixels. mode describes the type of frame buffer to use. eventMask specifies which events the window wishes to receive when active. Use the event constants described in Glow for the event mask.

virtual ~GlowWindow(void)

The destructor for GlowWindow removes the GLUT window and recursively deletes its entire hierarchy.

Toplevel window operations
You can perform special toplevel window operations using these methods.

void Maximize(void)

Attempts to maximize the window, and also removes the window decorations. The intent is to create a full-screen window. Note that the exact behavior is window manager dependent. Some window managers may not support removing the window decorations, and the resulting width and height of the window is not guaranteed to be the width and height of the screen. This operation can be undone by simply calling Reshape().

void Iconify(void)

Attempts to iconify the window, if supported by the window manager. If window iconification is not supported, does nothing.
