module glowQuickPalette

General information

The glowQuickPalette module contains classes that implement the QuickPalette interface for fast, easy creation of widget palettes.

Classes

base GlowQuickPalette

The base QuickPalette API, from which other QuickPalette containers inherit.

class GlowQuickPaletteWindow

A GLOW window containing a QuickPalette.

class GlowQuickPaletteSubwindow

A GLOW subwindow containing a QuickPalette.

class GlowQuickPanelWidget

A panel in a QuickPalette, useful for organizing and positioning widgets.

class GlowQuickRadioGroupWidget

A special type of radio group used in a QuickPalette.
