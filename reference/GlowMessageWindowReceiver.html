typedef GlowMessageWindowReceiver

General information

type: abstract
synonym: TReceiver<const GlowMessageWindowMessage&>
module: glowMessageWindow

GlowMessageWindowReceiver is the base class of receivers of message window events. Subclasses should override the OnMessage() method to handle an event.

GLOW programs that use message windows may use GlowMessageWindowReceivers to perform additional processing when a window is dismissed.

Methods

Destructor

~GlowMessageWindowReceiver(void)

Automatically informs any attached GlowMessageWindows that this receiver is going away.

Information on senders
A receiver can provide rudimentary information on which senders are bound to it.

unsigned int NumSenders(void)

Returns the number of message windows bound to this receiver.

unsigned int NumTrackers(void)

Returns the number of trackers bound to this receiver.

Overrideable methods

Receiving messages

virtual void OnMessage(const GlowMessageWindowMessage& message) = 0

This is a pure virtual method that must be overridden by a subclass. It is called whenever the receiver receives an event.
