typedef GlowScrollBarReceiver

General information

type: abstract
synonym: TReceiver<const GlowScrollBarMessage&>
module: glowScrollBarWidget

GlowScrollBarReceiver is the base class of receivers of scroll bar events. Subclasses should override the OnMessage() method to handle such an event.

Most GLOW programs that use scroll bars will use GlowScrollBarReceiver to handle scroll bar events.

Types

Types

typedef const GlowScrollBarMessage& MessageType

A synonym for the type of message.

Methods

Destructor

~GlowScrollBarReceiver(void)

Automatically informs any attached GlowScrollBarWidgets that this receiver is going away.

Information on senders
A receiver can provide rudimentary information on which senders are bound to it.

unsigned int NumSenders(void)

Returns the number of scroll bars bound to this receiver.

unsigned int NumTrackers(void)

Returns the number of trackers bound to this receiver.

Overrideable methods

Receiving messages

virtual void OnMessage(const GlowScrollBarMessage& message) = 0

This is a pure virtual method that must be overridden by a subclass. It is called whenever the receiver receives a scroll bar event.
